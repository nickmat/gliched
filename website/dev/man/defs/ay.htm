Calendar Description

The Astronomical Year calendar is a hybrid variant combining the Julian and the Gregorian calendars. The calendar contains the year zero and prior years are shown as negative.

See Wikipedia entry.

Use Case

  • This calendar is commonly used by astronomers. Typically only the year value is used.

Definition

The Record consists of four Fields named scheme, year, month and day.

Record - Default Order
schemeyearmonthday
Record - Ranking Order
yearmonthdayscheme
Historic Era

For the range past..2299160 (j# 4 Oct 1582) the Julian Calendar is used. There is zero and negative years. For the range (g# 15 Oct 1582) 2299161..future the Gregorian calendar is used.

The year 1582 has 355 days.

As a Hybrid calendar, it has no single epoch.

Script

Default Library Script
lexicon jg { name "Hybrid Julian and Gregorian scheme names"; fieldname scheme; lang en; pseudo Scheme, sch; tokens { 0, "Julian", "j"; 1, "Gregorian", "g"; } } grammar jg { fields scheme year month day; rank year month day; lexicons m jg; alias pseudo { dd day; mm month; yyyy year; s scheme; } alias unit { d day; m month; y year; s scheme; } format dmys "{day} |{month:m:a} |{year} |{scheme:jg:a}"; format "dmys+", "{day} |{month:m} |{year} |{scheme:jg}"; format dmy "{day} |{month:m:a} |{year}"; format "dmy+", "{day} |{month:m} |{year}"; preferred dmy; } scheme ay { name "Astronomical Year"; base hybrid { fields scheme year, month, day; scheme 0 { base julian; } change 2299161; // g# 15 Oct 1582 scheme 1 { base gregorian; } } grammar jg; }

The script uses the lexicon "m" definition from j Julian for the month field.
